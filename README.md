# E-Commerce Website
A full-stack e-commerce web application built with React, Node.js, Express, and MySQL.
Demo: https://basic-e-commerce-nine.vercel.app

## Features
- User authentication (JWT)
- Product listing & search
- Shopping cart
- Order management
- Admin dashboard
-

### Frontend

### Backend
